# Practice

В этой папке лежат заготовки `.cpp` и `.hpp` для самостоятельного решения задач.

Как работать:

1. Запустите сайт курса из папки `site`.
2. Откройте нужную задачу на сайте.
3. Найдите указанный файл в `practice`.
4. Напишите код вместо `TODO`.
5. Запустите этот файл отдельно в Visual Studio или через любой C++17-компилятор.
6. После самостоятельной попытки сравните решение с файлом из `solutions`.

Не нужно открывать старую `.sln`-структуру и собирать весь проект целиком. Текущий вариант курса устроен проще: сайт нужен для чтения и навигации, `practice` — для вашего кода, `solutions` — для сверки.
